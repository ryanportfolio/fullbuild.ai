# fullbuild.ai

> idea → design → engineering → shipped

Recruiter-facing portfolio showpiece and Case Study 00: the making of
fullbuild.ai itself. The site traces one intent through product decisions,
visual design, engineering constraints, and an honestly locked shipment gate.

## Local use

```powershell
npm run build
npm test
npm run verify
npm run serve
```

Open the localhost URL printed by `npm run serve`. The build is dependency-free:
Node.js emits semantic static HTML from `content/site.json`; CSS supplies the
responsive workshop; small ES modules add a guarded lifecycle state machine,
deterministic phase tokens, and a raw-WebGL2 artifact whose seeded,
shared-topology mesh morphs through the four manufacture states as you scroll.
The rendered layer only enhances: semantic DOM and inline SVG silhouettes
preserve the complete story when JavaScript, WebGL, motion, or pointer input
are unavailable. The local Anybody variable font is distributed under the SIL
Open Font License in `assets/fonts/OFL.txt`.

## Verification state

Contract tests and the deterministic verifier gate content, stage order,
shipment truth, phase laws, assets, and transfer budgets. Real-browser review
at desktop, tablet, and mobile sizes remains the acceptance gate for visual
craft, accessibility, and anti-slop resilience.

This repository is in the **engineering** stage. Production shipment remains
locked: no reachable deployment URL or release evidence exists. Owner identity,
résumé, contact channels, and external case studies are intentionally absent
until real source material is supplied.
