# Site copy voice

Rules for all user-facing copy on fullbuild.ai (pages, prototypes, UI strings, meta tags).

## Hard bans

- Em dashes (U+2014), anywhere: prose, headings, UI strings, titles. Use a period, comma, colon, parentheses, or " / " and " · " for label-style separators.
- Periods on headings and display text (h1-h6, kickers, section labels, big display strings). Sentence punctuation stays in body prose only.

These extend the global writing skill and were set by the owner on 2026-07-21; apply them even when the writing skill is not loaded.
