# Out-of-this-world web-design research

Date: 2026-07-20
Scope: fullbuild.ai visual identity, interaction, motion, placeholder resilience, and recruiter-facing memorability. Hosting, publishing, SEO/social cards, caching, and release plumbing excluded.

## Baseline finding

The current site is disciplined but belongs to a recognizable AI-era creative-development cluster: near-black field, warm white oversized grotesk, vermilion accent, wireframe orb, tiny uppercase metadata, and ruled editorial sections. The WebGL object demonstrates technique without becoming navigation, proof, or narrative. Mobile removes most of the signature from the first viewport.

A fresh browser baseline proposed “The Red Thread / Proof Press.” It improved subject connection but still preserved the same palette and proof language, avoided cinematic risk, supplied no placeholder portfolio architecture, and stopped at technical verification. This became the RED test for the new skill.

## What award-winning work actually shares

Research covered current Webby winners and recognized Awwwards/FWA/CSSDA work: Shopify Winter ’26, Antoine Wodniack, Locomotive, Justine Soulié, SMSY Gen-02, ERA, and Mark Woodland.

They do not share one style. They share one authored law:

- Shopify turns a product release into a complete Renaissance world; art, props, navigation, and transitions reinforce the same idea.
- Wodniack makes generative code and motion the identity and gives work immediate priority.
- Locomotive turns its name and progressive language into timing, type, and navigation behavior.
- Soulié makes the actual illustrations the interface; each project receives different behavior.
- SMSY makes the spatial world the portfolio navigation and the proof of real-time skill.
- ERA derives its recurring line, patterns, and interactive map from the architecture itself.
- Mark Woodland uses editorial motion to carry a specific argument and public identity.

Transferable lesson: spectacle works when removing it would damage meaning, navigation, or proof. Otherwise it is garnish.

## AI-tell taxonomy

Avoiding only purple gradients is obsolete. Current tells include whole clusters:

- structural repetition: centered hero, equal cards, alternating sections, identical stage layouts;
- category costumes: dark-neon dev, cream editorial, faux brutalism, glass SaaS;
- tech theater: mesh, particles, fake terminal, cursor follower, or shader unrelated to behavior;
- typographic defaults: familiar geometric sans, label forests, slash metadata, oversized type used as the only compositional move;
- motion sameness: every block fades up, everything moves continuously, no dramatic arc;
- language sameness: “transform/elevate/seamless/end-to-end,” symmetrical triples, defensive process copy;
- mobile retreat: signature removed and desktop simply stacked;
- fabricated specificity: fake metrics, logos, clients, testimonials, or status data.

The new rejection threshold is a cluster of four familiar tells in one viewport. The remedy is not novelty for its own sake; it is a design world generated from fullbuild’s actual transformation from intent to shipped artifact.

## Design imperatives for the rebuild

1. Replace the current visual thesis, not merely polish it.
2. Make one retellable mechanism carry identity, state, navigation, and proof.
3. Give idea/design/engineering/shipped visibly different physics within one lineage.
4. Show the signature inside the first 390×844 viewport.
5. Treat placeholder cases as intentional visual objects, never empty cards or fake claims.
6. Move from process defense toward confident authorship and recruiter orientation.
7. Build motion as arrival → transformation → rest → payoff.
8. Verify every major state in a real browser and continue until improvements become taste trades rather than clear upgrades.
