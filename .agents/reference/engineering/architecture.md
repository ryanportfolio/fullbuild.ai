# Architecture

Use a DOM-first static architecture. `content/site.json` owns factual copy and honest placeholder states. `src/template.mjs` emits the complete semantic narrative, navigation, lifecycle evidence, and locked shipment state into `index.html`. CSS owns the Build Seam world, responsive composition, four material representations, reduced motion, forced colors, and print.

Small dependency-free ES modules own only deterministic phase tokens, the guarded lifecycle state machine, direct seam manipulation, announcements, and reveal timing. No essential meaning or action may depend on JavaScript, pointer input, animation, canvas, or a rendering engine.

One lifecycle identity moves through four distinct material laws: fluid Idea, planar Design, articulated Engineering, and fused-but-gated Shipped. `deployment.verified === false` prevents the shipped state from activating and announces the missing production proof.
