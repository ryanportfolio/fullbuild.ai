# Technology stack

- Semantic HTML5 generated from canonical JSON content.
- Modern CSS for the complete visual and responsive system.
- Dependency-free JavaScript ES modules for lifecycle state and direct manipulation.
- Node.js built-in test runner for content, phase-token, state-machine, accessibility, and generated-page contracts.
- Anybody variable type family, vendored from the official Google Fonts repository under the SIL Open Font License. Its `wdth` and `wght` axes are lifecycle material parameters.

Avoid framework, component-library, animation-library, canvas, WebGL, and 3D-engine dependencies unless measured evidence proves the native DOM/CSS implementation cannot satisfy the contract.
